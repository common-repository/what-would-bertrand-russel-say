Bertrand Russel quotes for the daily round

== Description ==

This is not just a plugin, it symbolizes the doubt and  disillusionment of an entire generation summed up in the quotes mostly put forth by Bertrand Russel.  When activated you will randomly see a quote from <cite>Bertrand Russel</cite> in the upper right of your admin screen on every page.
